# Specification

## Summary
**Goal:** Enhance the attack path visualizer with new interaction capabilities including bidirectional arrows, text rotation, free transform tools, and updated icon representations.

**Planned changes:**
- Add a bidirectional vertical arrow icon to the toolbar that can be dragged and placed on the canvas
- Enable text label rotation with rotation controls
- Implement a free transform tool that allows moving and resizing both icons and text labels with visual handles
- Change the .exe icon from a graphical representation to text displaying '.exe'
- Change the Excel icon to text displaying 'Microsoft Excel'
- Change the Word icon to text displaying 'Microsoft Word'
- Remove the IP Address icon from the toolbar

**User-visible outcome:** Users can place bidirectional arrows on diagrams, rotate text labels to any angle, resize and reposition elements using a free transform tool, work with text-based representations for .exe, Excel, and Word elements, and the IP Address icon is no longer available in the toolbar.
