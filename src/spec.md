# Specification

## Summary
**Goal:** Enhance the attack path visualizer with diagram naming, selection, clearing, PNG export, improved transform tool, and remove bidirectional arrow.

**Planned changes:**
- Add diagram naming functionality when saving attack path diagrams
- Implement diagram selection interface to choose from multiple saved diagrams by name
- Add a clear button to remove all elements from the canvas
- Replace JPEG export with PNG export functionality
- Fix the free transform tool to properly move and resize icons and text labels
- Remove the bidirectional arrow icon from the toolbar
- Update backend diagram storage to support diagram names

**User-visible outcome:** Users can name and select saved diagrams, clear the canvas with one click, export diagrams as PNG files, properly transform icons and text labels, and work without the bidirectional arrow icon.
